hexo图片不展示

图片不展示时，观察图片src的路径是否有误。曾出现过在列表全展示时无法查看图片，点进具体博文后可展示图片的异常。

参考地址:https://www.jianshu.com/p/db02d775aed0

文章大意是说,使用 npm install https://github.com/CodeFalling/hexo-asset-image –save 安装0.0.5版本的hexo-asset-image插件。 使用 npm install hexo-asset-image –save 安装的是1.0.0版本的hexo-asset-image插件。 两者最直接的区别是映射关系不同。

也就是因为插件版本更新了，之前的使用方法有所变化，于是我安装了0.0.5版本的插件，就成功加载图片了.
